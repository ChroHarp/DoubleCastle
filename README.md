# DoubleCastle

單頁版瀏覽器棋盤遊戲，支援雙人對戰與人機對戰，棋盤大小可調（1∼12）。

## 玩法
1. 選擇是否啟用「電腦對戰」。
1. 若啟用電腦對戰，可選擇玩家是否先手。
1. 調整棋盤大小後開始對局。
1. 輪到你時，點選一個「合法」的空格落子。
1. 若場上已無任何合法落子位置，遊戲結束，最後落子的一方獲勝。

## 合法落子規則
- 落子位置必須是空格。
- 同一「行」已存在的棋子數量 + 同一「列」已存在的棋子數量 < 2。
- 不符合規則的位置會顯示禁止符號且不可落子。

## 操作
- 「復原」：可回到上一手（僅一步）。
- 「重新開始」：重置棋盤與狀態。
- 「全螢幕」：切換全螢幕模式。

## 執行
直接用瀏覽器開啟 `index.html` 即可遊玩，無需安裝或打包。
